HTTP 首部字段是构成 HTTP 报文的要素之一。在客户端与服务端之间以 HTTP 协议进行通信的过程中，无论是请求还是响应都会使用首部字段，它能够起到传递额外重要信息的作用。


## 4种 HTTP 首部字段类型

HTTP 首部字段根据实际用途被分为以下4种类型。


###  通用首部字段
    请求报文和响应报文两方都会使用的首部
